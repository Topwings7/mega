기능시연 순서



1. 메인페이지
   - 전체적으로 살짝 보여주며 고객들은 로그인 후 영화예매,영화정보,게시판 등을 이용할 수 있다는 간략 설명

2. 회원가입
   - 회원가입시 input박스 디자인. 파일첨부 버튼 디자인 등 css 간략설명 후 ajax활용하여 비밀번호와 아이디 확인
   - 회원 정보 아이디, 패스워드, 이메일, 를 AJAX처리로 중복및 형식 검증.
   - 주소는 다음 API 사용 신주소 검색 입력 가능.
   - 사진 첨부 안할시에 기본 사진

3. 로그인
   - 로그인시 아이디 또는 패스워드가 불일치할 경우 보여주기

4. 회원정보.(수정)
   - 회원 정보 아이디, 패스워드, 이메일, 를 AJAX처리로 중복및 형식 검증.
   - 주소는 다음 API 사용 신주소 검색 입력 가능.
   - 사진 첨부 안할시에 기본 사진

6 회원 정보 찾기 시연.
  아이디찾기 시연
 패스워드 찾기 시연 (네이버로그인 메일발신 확인)

- 로그인 해서



7. 영화정보
   - 리스트 출력되는것 설명.(인기순, 상영예정, 영화DB...)

   - 상세정보 클릭시 모달창을 이용해서 데이터 주고받는걸 강조하기.

   - 누적관객, 평점, 후기, 좋아요~!
     (1)누적관객: 고객이 영화를 예매하면 누적관객수 증가.
     (2)평점: 별점을 통해 영화의 평점이 구분되어 인기순에 영향을끼침.
     (3)후기: 고객은 영화에대한 후기를 작성 가능하며, 후기글의 페이징 처리는 ajax를 이용.
     (4)좋아요: 고객은 영화에 좋아요 버튼을 클릭가능하며, 한 번 클릭후 두번 째 클릭할 시 좋아요가 취소된다.

   - 영상 재생.

          
8. 게시판 이용.
   - 게시판 종류는 3가지 자유게시판 문의 게시판 공지 게시판(공지게시판 기능은 관리자기능 시연시 보여준다.)
   - 공지는 아코디언 목록으로 처리
   - 글 쓰기, 글 수정, 글 삭제, 댓글, 페이징 보여주기.

9. 영화예매.
   - 회원정보에서 포인트와 누적금액 할인율 등을 먼저 보여주고 예매 시작.
   - 영화 시간표 출력: 시간이 지나면 영화예매가 불가능으로 클릭이 안 된다.
   - 예매 진행.
     (1) 인원수가 선택이 안 되면 좌석 선택 불가능.
     (2) 좌석선택시 하단에 선택인원수/총인원수 변경 됨을 보여주기.
     (3) 좌석및 인원수 선택시 오른쪽 하단에 선택 정보및 금액이 보여짐.
     (4) "다음"버튼 누를 시 로그인이 안되어 있거나, 선택인원수와 총 인원수가 맞지 않으면 진행이 안 됨을 보여주기.
     (5) "다음"으로 넘어가면 고객이 선택한 예매정보가 출력이되고, 고객의 현재 포인트 및 할인율 누적 포인트 정보가 나옴.
     (6) 예매 진행은 "예매","포인트"로 나누어지고, 포인트로 구매시에는 포인트만 차감 예매로 구매시에는 누적금액증가 및 포인트 증가.
     (7) 예매가 완료되면 회원정보에서 누적금액 및 포인트증가or차감, 등급변경 보여주기.
     (8) 회원정보에서 예매 취소진행시 시간이 지난 영화는 예매 취소가 불가능하고, 아직 상영시작 전 영화는 예매 취소 가능함을 보여주기. (AJAX 처리로 부분 데이터 처리)
     (9) 예매 취소시 누적금액,포인트 차감 보여주기.

10.회원탈퇴.


관리자.

0. 관리자 등록을 먼저 시연.
 - 등록한 관리자로 로그인 후 아래 기능을 시연.

1. 영화등록.
   영화등록 
   - 영화 등록 : 영화 등록시 이미지 1개 필수, 비디오 입력은필수
   - 장르선택시 장르코드는 영화 코드와 장르에 쓰이는 데이터가 된다. (영화 코드 = 장르코드 + 시퀀스)
   - 이미지 미선택 입력시 기본이미지 적용 (오라클 default 필드)

2. 영화수정.
  - 이미 등록되어 있던 영화 정보가 먼저 필드에 출력 되고 수정 가능.

3. 영화삭제.
  - 영화는 바로 삭제되어 카드가 사라진다.

-- 스토어 상품 등록만 보여줌.

4. 공지 게시판 기능 시연
   - 출력은 아코디언으로 출력됨을 시연
   - 공지 등록, 첨부 기능 설명.
   - 공지 삭제 기능 시연
